# 佐藤 将高 / Masataka Sato

## About Me

- 福島県福島市出身
- 小学生の頃からHTMLでゲーム攻略サイトを制作し、ものづくりに没頭
- 東京大学大学院 情報理工学系研究科 創造情報学専攻 修了（自然言語処理・データマイニング）
- グリー株式会社にてフルスタックエンジニアとして勤務
- 2016年6月 ファインディ株式会社を共同創業、取締役CTOに就任
